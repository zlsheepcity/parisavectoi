Entrée / enter

The project is dedicated to the process of creation. In fact you are visiting custom web-gallery of holiday trip into the city of Paris. In the background there is a path of creation where each step does matter. This including background story, involved people, used technologies and even the art of coding.

This one is the experiment to display and test an idea: To achieve magical results, try to maintain the magic at every stage of creation.

La source / the source

Paris, Summer 2025

The city of Paris was chosen as proven source of inspiration. It is a great capital of european culture filled with different types of art and artists.

Feelings and emotions gathered during the short trip were converted into memories and some media.

Les images / the pictures

In addition to project main idea, this section brings its own message: Each photo [on your film or card] does matter to translate full story feelings.

Presented photos taken on Canon S95 digital camera.

La typographie / typography

The web typeface used in this project — is Voklron designed by Friedrich Althausen.

The typography layout was inspired by Edward Tufte books and paris galleries.

You may find related links in references section to get more interesting and beautyful details.

Le code / the code

All gathered from the trip — including digital media, inspiration, motivation, ideas — were converted into web page code.

Imagine if you got some positive vibes from the previous sections. All this feelings are now just a pile of symbols. Beeing converted into code, this feelings could be reproduced, copied, changed and even updated to the next level.

In 2025 the project code was hosted and provided by gihub.com service. You may find the full code repository link in references section or just enjoy the selected gallery below.

Le résumé / summary

It was typical art project in web page format. With beautiful pictures, illustrations, philosophycal notes and creative ideas.

The project target is a web page viewer. Literally You are the target now.

The project goal is achieved if You got any kind of fresh ideas for yourself. The source of this new ideas was located in Paris even if you werent there. Magic, yes!
